西江千户苗寨

关于古老苗族村落的文化

西江千户苗寨是苗族村寨中规模最大、保存最为完整的一处，被誉为“中国最美十大民族风情村寨”之一。一排排的木质建筑错落有致地分布在清山绿水之间，有栋房子都有着独特的风格和形状。在苗寨里可以听到苗族人民奇妙的歌声和舞蹈，品尝到地道的苗家美食，更能深入了解苗族古老的传统文化。西江千户苗寨主要景点有西江苗族博物馆、鼓蔵头家、活路头家、酿酒坊、刺绣坊、蜡染坊、银饰坊、观景台、噶歌古道、田园观光区等

苗族吊脚楼

上一步，望宝梁，一轮太阳在中央，一元行始呈祥瑞。上二步喜洋洋，乾坤二字在两旁，日月成双永世享...

吊脚楼，多依山就势而建，整体布局讲究背山面水。吊脚楼往往分为三层，楼下安放农具、堆放柴草；中间一层可用作卧室、堆放粮食等；最上面一层为姑娘楼，是土家姑娘绣花、读书的地方。上面两层窗外有绕楼的木栏走廊，可用来晾晒衣物、谷物等。吊脚楼底部支撑的柱子直且长，屋顶则飞檐走角。苗族吊脚楼还具有重要的社会功能，它是苗族人民生活、工作、文化交流的重要场所，也是苗族传统文化的载体之一。在吊脚楼中，苗族人民世代相传的习俗、传统手工艺、歌舞文化等得以传承和发展。

苗族风俗文化

苗族的无酒不成礼，长桌宴

苗族宴席的最高形式与隆重礼仪，苗族人普遍喜欢喝酒，常以酒解除疲劳，以酒示敬，以酒传情，饮酒为乐。苗家的酒主要有白酒、甜酒、刺梨酒、泡酒等。白酒即土酒，苗家人常饮，以大米、糯米、玉米、高粱等为原料酿制。 "无酒不成礼"已是他们遵守的礼仪，酒是待客议事、婚丧嫁娶、起房建屋、逢年过节的必备品.苗族人还极为讲究待客的饮酒礼节，如主人向客人敬酒要敬两杯，谓"两条腿走路"，敬酒者双手端杯，按正、反方向敬两周，敬到最后一人时，双方交杯对饮；又如以牛角杯敬酒，客人如不喝完，则"杯"倒酒淌，使人不能不喝。
