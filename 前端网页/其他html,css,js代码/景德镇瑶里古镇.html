历史文化

瓷都之乡——“家家窑火，户户陶埏”

瑶里，古名“窑里” 远在唐代中叶，这里就有生产陶瓷的手工业作坊，因瓷窑出名而得名。直到本世纪初，瓷窑外迁，“窑里”才改名为瑶里。 瑶里，古名“窑里”，因是景德镇陶瓷发祥地而得名，远在唐代中叶，这里就有生产陶瓷的手工作坊。瑶里位于举世闻名的瓷都东北端，地处三大世界文化遗产(黄山、庐山、西递和宏村)的中心，素有“瓷之源，茶之乡，林之海”的美称。

雨过天晴云破处，瑶里即梦里

天青色等烟雨，而我在等你

一城山水诗画，千年古镇瑶里。瑶里古镇不仅在建筑上独具特色，还因其丰富的自然资源而被誉为“林之海”，古镇四周被群山环抱，瑶河穿镇而过，形成了宁静清澈的自然景观。 此外，瑶里古镇也是中国重要的茶叶产地之一，被称为“茶之乡”，白居易的《琵琶行》中“前月浮梁买茶去”说的正是瑶里所在的浮梁县。 古镇的建筑风格独特，保存大量明清古建筑，以徽派建筑为主，青砖黑瓦、马头墙、飞檐翘角，具有浓厚的地方特色。

时光似水，流云舒卷，青山不老，古村千年

瑶里改编纪念碑

瑶里改编纪念碑，瑶里是著名的皖赣边区三年游击战争的根据地之一。1937年七·七事变后，中国共产党将南方八省十四个地区的红军游击队改编成国民革命军新编第四军，这支部队离开皖赣边区根据地，奔赴了抗日线，留下的部分人员，设立了新四军驻瑶里留守处，处理善后事宜，留守处旧址，已列为市级保护文物。
