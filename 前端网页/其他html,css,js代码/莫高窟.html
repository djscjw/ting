莫高窟

前世古今

莫高窟，又被称为敦煌石窟，是中国四大石窟之一，位于甘肃省敦煌市的鸣沙山脚下，毗邻大漠。

公元前209年

公元前209年，匈奴击败月氏，占领河西走廊。公元前121年，霍去病打败匈奴，将河西纳入汉朝的版图。 公元前111年，西汉在河西设置武威、酒泉、张掖、敦煌四郡。从此，敦煌开始繁荣，丝绸之路也得以开通。 敦煌是佛教传入中原的第一站，也是西域僧人进入中原的第一站。僧人在敦煌开凿禅窟，信佛的人越来越多，各色人都在开凿石窟做功德，石窟越来越多，莫高窟酝酿着博大的佛教文化。 隋文帝隋炀帝都特别信佛，隋朝37年，莫高窟完成94个石窟。敦煌地处中西商路的要道，往来客商一路上不平安，也需要在敦煌祈求佛的保佑.公元633年，敦煌更名为沙洲，随着盛唐的到来，敦煌迎来最辉煌的时代。

公元755年

公元755年，安史之乱爆发，河西防御空虚，公元786年，敦煌为吐蕃占领，赤松德赞大力弘扬佛法，敦煌佛教得到进一步发展。公元848年，敦煌人揭竿而起，赶走吐蕃奴隶主，和长安联络，再度回到唐王朝的怀抱，但唐王朝已经走向衰落。公元1006年，新疆与中亚地区的黑汗王朝占领于阗王国之后，继续向东，敦煌三界寺的僧人将多年积累的大量经卷、文书、绢画藏进了敦煌17号石窟，这批文物从此冬眠。公元1036年，西夏占领沙洲，统治达192年，这是敦煌成立以来，由少数民族统治最长的时期。公元1715年，康熙收复敦煌，大批移民到敦煌屯田，敦煌社会经济开始恢复。1723年，复立沙洲所，清王朝恢复了对敦煌的统治。敦煌的经济政治文化得到重建与发展，但汉唐时期的辉煌一去不复返。清王朝走向穷途末路之时，丝绸之路成为历史，敦煌逐渐被遗忘。

1900

王圆箓与所雇敦煌贫士杨某在清理沙土时，意外发现莫高窟的藏洞，里面积满写卷、印本、画幡、铜佛等珍贵文物，这一发现为研究敦煌艺术、历史和宗教提供了宝贵的实物资料。然而20世纪初，这批刚发现的文物和文化遗产，被西方列强疯狂劫掠，文物流散于全世界十多个国家，藏于中国的仅占冰山一角。 藏经洞的发展震惊全世界，从此国际上多了一门显学——敦煌学。敦煌学是指以敦煌遗书、敦煌石窟艺术、以及敦煌史地为研究对象的一门学科。

现在

陈寅恪先生说过：“敦煌者，吾国学术之伤心史也。” 目前大约有16000多件敦煌艺术主要珍藏在中国国家图书馆 敦煌文化的高峰已经不再，汉唐时代的辉煌不会重现，但敦煌的历史还要写下去，对于中华儿女来说，继承它博大的精神内涵最为重要，敦煌对我们意味着什么，毫无疑问，敦煌是我们的文化血脉，敦煌就是我们自己。在这里我们也要致敬那一代代付出的敦煌研究者们。莫高人身上那份“择一事、终一生”的信念，那种“夜夜敦煌入梦来”的炽热情怀，那份“做好传帮带、传好接力棒”的担当，始终在赓续绵延。
